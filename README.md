# Instant Tooltip+

A mod for S.T.A.L.K.E.R Anomaly that shows tooltips instantly, and hides them when a key (**LCtrl**) is pressed.

Also allows you to hide tooltips by default by pressing **RCtrl**.

### Installation

1. Drag and drop `gamedata/` to your `Anomaly/` folder.
2. Profit.
